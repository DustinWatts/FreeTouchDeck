Upload a new logo

You can cumstomize the logo's that are used. If you upload a file with a name that already exists, the file is overwritten. You can only upload .bmp images. There images should be 75x75 pixels and be in a 24-bit RGB format. A perfectly black (#000000) pixel will be rendered as transparent.
